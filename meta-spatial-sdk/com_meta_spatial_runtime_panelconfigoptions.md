# PanelConfigOptions Class

Extends
[PanelShapeConfig](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_panelshapeconfig)
*Modifiers:
final*

Configuration options for the panel. This is used to configure the panel's size, shape, resolution, mesh creator, and other properties. This is used to create the panel's [SceneObject](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_sceneobject) .

More information on Panel creation and registration can be found [here](https://developers.meta.com/horizon/documentation/spatial-sdk/spatial-sdk-2dpanel-registration) .

## See Also

- [PanelSceneObject](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_panelsceneobject)
