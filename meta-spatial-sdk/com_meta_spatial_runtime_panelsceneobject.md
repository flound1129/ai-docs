# PanelSceneObject Class

Extends
[SceneObject](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_sceneobject)
*Modifiers:
final*

PanelSceneObject combines several components to create an interactive 2D surface in 3D space in the following order:

- A [PanelSurface](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_panelsurface) that holds the pixels of the panel - A [PanelDisplay](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_paneldisplay) that renders content on the surface - A [PanelInputListener](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_panelinputlistener) that handles input events - A [PanelShape](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_panelshape) that defines the geometry (quad, cylinder, etc.)

Example: Retrieving a PanelSceneObject from an entity using SceneObjectSystem

```kotlin // Get the SceneObject associated with the entity (returns a CompletableFuture)
systemManager.findSystem<SceneObjectSystem>().getSceneObject(entity)?.thenAccept { sceneObject ->
    if (sceneObject is PanelSceneObject) {
        // Now you can work with the panel
    }
} ```

This method updates the dimensions and local offset of the IsdkPanelDimensions component based on the panel's shape configuration.
