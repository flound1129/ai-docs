# SecondaryCard Function

*Modifiers:
final*

Secondary card with subtle background, suitable for supporting content. Provides a more subdued visual treatment than primary cards.

## Parameters

modifier:
Modifier
Modifier to be applied to the card
onClick:
Function0?
Optional callback to be invoked when the card is clicked
contentPadding:
Dp
Padding to be applied to the card content, defaults to medium spacing
content:
Function1
The content to be displayed inside the card
