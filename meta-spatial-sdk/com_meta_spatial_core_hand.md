# Hand Enum

Represents the hand side in a spatial context.

This enum class is used to distinguish between the left and right hand, typically in applications involving spatial interactions or gestures.

## Signature

```kotlin
enum Hand : Enum<Hand>
```

## Enumeration Constants

| Member | Description |
| --- | --- |
| LEFT | An Int value of zero (0) specifies the left hand |
| RIGHT | An Int value of one (1) specifies the right hand |

## Properties

| **side** : Int [Get] | An integer representing the side of the hand, where 0 is LEFT and 1 is RIGHT. Signature ```kotlin val side: Int ``` |
