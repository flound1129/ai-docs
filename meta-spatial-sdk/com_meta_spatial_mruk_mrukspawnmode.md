# MRUKSpawnMode Enum

Defines how MRUK objects should be spawned across rooms in a mixed reality environment.

## Signature

```kotlin
enum MRUKSpawnMode : Enum<MRUKSpawnMode>
```

## Enumeration Constants

| Member | Description |
| --- | --- |
| NONE | No objects will be spawned |
| CURRENT_ROOM_ONLY | Objects will only be spawned in the current room |
| ALL_ROOMS | Objects will be spawned in all available rooms |
