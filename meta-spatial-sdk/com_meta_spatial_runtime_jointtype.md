# JointType Enum

Enumeration of tracked body joints supported by the Spatial SDK.

Notes:

- Each entry’s index matches the native joint index expected by the runtime. - These values map directly to the corresponding OpenXR joint identifiers where applicable, following the layout used by the underlying OpenXR extensions (e.g., FB body/hand tracking). - Use with body tracking APIs (e.g., allocateBodyTrackingBuffers/updateBodyTrackingBuffers) to interpret pose/flag arrays by index.

## Signature

```kotlin
enum JointType : Enum<JointType>
```
