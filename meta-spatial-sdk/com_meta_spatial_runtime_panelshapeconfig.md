# PanelShapeConfig Class

*Modifiers:
open*

This class defines the common properties used to configure the geometry and appearance of panels in 3D space. It serves as the base class for [PanelConfigOptions](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_panelconfigoptions) , which provides additional parameters to intialize the Android Virtual Display.

