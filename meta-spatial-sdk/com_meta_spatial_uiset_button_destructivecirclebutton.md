# DestructiveCircleButton Function

*Modifiers:
final*

## Signature

```kotlin
fun DestructiveCircleButton(icon: () -> Unit, onClick: () -> Unit, isEnabled: Boolean = true)
```

## Parameters

icon:
Function0
The icon to display on the button.
onClick:
Function0
The action to perform when the button is clicked.
isEnabled:
Boolean
