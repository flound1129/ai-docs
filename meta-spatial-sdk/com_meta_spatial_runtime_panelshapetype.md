# PanelShapeType Enum

Defines the types of panel shapes available.

Each shape type determines how the panel is rendered in 3D space.

## Signature

```kotlin
enum PanelShapeType : Enum<PanelShapeType>
```

## Enumeration Constants

| Member | Description |
| --- | --- |
| QUAD | Flat rectangular panel |
| CYLINDER | Curved panel that follows a cylindrical surface |
| EQUIRECT | Spherical panel that covers a full 360° view |
| EQUIRECT180 | Spherical panel that covers a 180° view |
