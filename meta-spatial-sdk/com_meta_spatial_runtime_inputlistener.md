# InputListener Interface

Interface for handling input events on [SceneObject](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_sceneobject) s.

InputListener provides a comprehensive set of callback methods for responding to various input events such as clicks, hovers, and general pointer interactions.

When [IsdkFeature](/reference/spatial-sdk/v0.10.1/com_meta_spatial_isdk_isdkfeature) is enabled, you should use onPointerEvent instead of onInput. More info [here](https://developers.meta.com/horizon/documentation/spatial-sdk/spatial-sdk-isdk-listen-to-input-events) .

[SceneObject](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_sceneobject) can have multiple InputListeners attached, allowing for modular input handling where different components can respond to the same input events independently.

Example usage:

```kotlin // Create a custom input listener
val myInputListener = object : InputListener {
    override fun onClick(receiver: SceneObject, hitInfo: HitInfo, sourceOfInput: Entity) {
        // Handle click event
    }
}
// Attach the input listener to the SceneObject
val sceneObject = systemManager.findSystem<SceneObjectSystem>().getSceneObject(entity)
sceneObject?.thenAccept { so ->
   so.addInputListener(myInputListener)
} ```

## Functions

| **onClick** ( receiver , hitInfo , sourceOfInput ) | Called when a "click" is released while pointing at the object. This is a trigger release for controllers, and a pinch release for hands. This method is invoked at the end of a click interaction, when the user releases a button that was pressed while pointing at the object. It's paired with [InputListener.onClickDown](/reference/spatial-sdk/v0.10.1/com_meta_spatial_runtime_inputlistener#onclickdown) , which is called when the button is initially pressed. | **onInput** ( receiver , hitInfo , sourceOfInput , changed , buttonState , downTime ) | Called when any input event occurs on the object. Hovering itself is considered an input event, but pressing and holding a button down will send input to the last focused object.
