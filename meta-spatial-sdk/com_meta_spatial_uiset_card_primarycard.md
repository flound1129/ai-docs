# PrimaryCard Function

*Modifiers:
final*

Primary card with elevated appearance, suitable for main content areas. Provides the most prominent visual treatment with shadow and background.

## Parameters

modifier:
Modifier
Modifier to be applied to the card
onClick:
Function0?
Optional callback to be invoked when the card is clicked
contentPadding:
Dp
Padding to be applied to the card content, defaults to medium spacing
content:
Function1
The content to be displayed inside the card
